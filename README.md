# EV-Charging-Station-Dashboard
This is a Power BI intelligence dashboard titled "EV-Charging-Station-Analysis" — a 5-page interactive report built on a master dataset of EV charging transactions, station data, vehicle types, and charger categories. It transforms raw structured data into actionable business intelligence.
